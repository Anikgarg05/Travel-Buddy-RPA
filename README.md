# ✈️ Travel Buddy RPA – Flight Booking Automation

## 📌 Description
This project is a Robotic Process Automation (RPA) solution built using UiPath that automates the process of flight booking on the MakeMyTrip website.

The bot mimics human actions such as opening the website, entering travel details, searching for flights, and extracting relevant flight information.

## 🚀 Features
- Automatically opens MakeMyTrip website
- Enters source and destination cities
- Selects travel date
- Searches available flights
- Extracts flight details (price, timing, airline)
- Handles basic popups and delays

## ⚙️ Tech Stack
- UiPath Studio
- RPA (Robotic Process Automation)
- Web Automation
- Excel (optional for storing results)

## ▶️ How to Run
1. Open the project in UiPath Studio
2. Update input values (source, destination, date) if required
3. Run `Main.xaml`

## 💡 Use Case
This project demonstrates how repetitive web tasks like flight searching and booking can be automated using RPA, saving time and reducing manual effort.
